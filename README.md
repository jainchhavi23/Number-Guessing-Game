# 🎯 Number Guessing Game (Python)

A simple and fun **console-based number guessing game** built using **Python 🐍**.  
Choose a difficulty level, guess the hidden number, and try to win within limited attempts!

---

## ✨ Features

✅ Random number generation 🎲  
✅ Three difficulty levels  
✅ Limited attempts per level ⏳  
✅ Input validation & error handling ⚠️  
✅ Replay option 🔁  
✅ Beginner-friendly Python project  

---

## 🕹️ Difficulty Levels

| Level   | Number Range | Max Attempts |
|--------|--------------|--------------|
| 🟢 Easy   | 1 – 100      | 7            |
| 🟡 Medium | 1 – 500      | 10           |
| 🔴 Hard   | 1 – 1000     | 12           |

---

## 📌 How to Play

1. Run the Python file ▶️  
2. Enter `start` to begin the game  
3. Choose a level:
   - `easy`
   - `medium`
   - `hard`
4. Guess the number within the allowed attempts  
5. Hints will guide you:
   - 📉 **Too Low**
   - 📈 **Too High**
6. Win 🎉 or replay 🔁
